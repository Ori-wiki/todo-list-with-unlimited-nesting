# Todo List With Unlimited Nesting

Небольшое веб-приложение для работы с древовидным списком задач и элементов с неограниченной вложенностью. Проект позволяет создавать корневые и дочерние элементы, редактировать название, количество и сумму, а также удалять элементы вместе со всеми вложенными пунктами.

Демо: https://todo-list-with-unlimited-nesting.vercel.app/

## Возможности

- создание корневых и дочерних элементов
- редактирование данных прямо в таблице
- удаление элемента вместе со всей вложенной веткой
- отображение структуры в виде дерева
- интерфейс на React, TypeScript и Tailwind CSS

## Технологии

- React
- TypeScript
- Vite
- Tailwind CSS

## Запуск локально

```bash
npm install
npm run dev
```
